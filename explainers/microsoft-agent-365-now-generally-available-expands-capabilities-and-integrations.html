Microsoft Agent 365: A Control Plane to Govern Your AI Agents

AI agents are already in your environment — and they proliferate fast, span apps and clouds, and often run outside the visibility of the teams accountable for risk. Agent 365 is the control plane to observe, govern, and secure them.

The idea in five lines

  • The problem isn’t that agents exist — it’s that they multiply fast, span apps/endpoints/cloud, and act outside the control of risk teams.
  • An agent that can invoke tools, access data, and talk to other agents can turn a “helpful” workflow into data oversharing or over-privileged actions in seconds.
  • Agent 365 is a single control plane to observe, govern, and secure agents — both Microsoft-built and ecosystem/third-party.
  • New capabilities discover shadow AI agents (via Defender + Intune), apply policy guardrails, and sync inventories across AWS Bedrock and Google Cloud.
  • Governance principle: “You can’t govern what you can’t see, and you can’t secure what you don’t understand.”
The concept

A control plane for agent sprawl

As organizations move from pilot to adoption, agents show up everywhere — some acting on a user’s behalf with delegated access, others operating with their own credentials behind the scenes. Agent 365 governs both, using the admin and security workflows teams already run.

Figure 1 · The three jobs of the control plane
👁

Observe

End-to-end visibility of every agent — where it runs, what it can reach, and how it behaves — even as the count keeps changing.

⚖

Govern

Inventory and ownership, right-sized (least-privilege) permissions, and lifecycle controls across Microsoft and third-party agents.

🛡

Secure

Policy guardrails, runtime blocking of risky behavior, and network controls to stop prompt-based attacks and data exfiltration.

The through-line: you can’t govern what you can’t see — so discovery and observability come first.
What’s new

From shadow AI to cross-cloud control

Discovery

Find shadow AI agents

Defender + Intune discover unmanaged local agents (starting with OpenClaw, expanding to GitHub Copilot CLI and Claude Code) and let IT block how they run.

Context

Blast-radius mapping

Defender maps each agent’s devices, MCP servers, identities, and reachable cloud resources — so teams can assess exposure and set custom detections.

Cross-cloud

Registry sync

Discover and inventory agents across AWS Bedrock and Google Cloud, with basic lifecycle governance (start/stop/delete) coming.

Runtime

Guardrails & network controls

Entra network controls restrict agents to approved destinations, filter risky file movement, and help block malicious prompt-based attacks.

Sandbox

Windows 365 for Agents

A secured, managed Cloud PC environment purpose-built for agentic workloads, run under the same identity and management controls as employees.

Ecosystem

Third-party agents

Partner agents (Genspark, Zendesk, Zensai, and more) come pre-configured to be observed, governed, and secured — no integration work.

The governance checklist

What “governing agents” actually means

Microsoft’s partner services frame the practical governance work into five buckets — a useful checklist for any team standing up agent governance:

AreaThe question to answer
Inventory & ownershipWhat agents exist, who owns them, and where do they run?
Least privilegeAre permissions right-sized and access guardrails enforced — without slowing delivery?
Compliance & dataAre we preventing oversharing and producing audit-ready evidence?
Threats & multi-platformDo we understand attack paths and govern across vendors and clouds?
Ongoing operationsIs there lifecycle management, monitoring, and continuous governance hygiene?

“Enterprises can easily build AI agents today, but scaling them with trust and governance is where most initiatives stall.” — Raj Koneru, CEO, Kore.ai

Availability

Agent 365 is generally available for commercial customers — included in Microsoft 365 E7 or standalone at USD 15 per user per month. Several new discovery, context-mapping, and control capabilities are rolling out in public preview through June 2026.
